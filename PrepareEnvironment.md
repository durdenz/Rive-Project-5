# Prepare Environment for Application
## Update Node and NPM to current versions
1. Update Node
```bash
sudo npm cache clean
```
```bash
sudo npm install -g n
```
```bash
sudo n stable
```
2. Update npm
```bash
sudo npm install -g npm@latest
```

## Install Express
```bash
sudo npm install -g express-generator
```
